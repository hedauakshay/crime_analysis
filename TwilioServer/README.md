# Twilio Server
>     npm install twilio
>     Independent NodeJS Express project running on 1337
>     Run the Server
>     Use ngrok for tunelling
>     Open a new terminal
>     ./ngrok htttp 1337
>     Copy Forwarding URL
>     Open Twilio Console [Link: https://www.twilio.com/console]
>     Paste Forwarding URL in Console for receiving message
>     Setup is done
